### 📡 Privacy-Preserving Ambient Space Utilization Monitor
`IoT` `Arduino` `C++` `Sensor Fusion` `Edge Computing`

A real-time room occupancy counter built on Arduino — **no cameras, no cloud, zero PII collected**. Uses ultrasonic distance sensing as a doorway tripwire combined with PIR infrared verification to derive occupancy state anonymously at the edge.

- 🔒 **Privacy-by-design** — derives complex environmental states with zero personally identifiable data
- ⚡ **Non-blocking execution** via `millis()`-based timing — all sensors poll concurrently, no `delay()` bottlenecks
- 🎯 **2.5 s software debouncing** eliminates multi-count flutter from slow-moving objects
- 🔔 **Capacity alerts** via buzzer + LED when occupancy reaches threshold; 5-min idle auto-reset corrects drift
- 📊 Live 4-digit display + serial dashboard reporting count, %, uptime, and entry/exit totals

### 🩺 Blood Glucose Forecasting System
`Python` `scikit-learn` `Pandas` `ML` `Healthcare`

A machine learning pipeline for short-term blood glucose prediction in Type 1 Diabetes patients, trained on the real-world AZT1D dataset (25 patients, 5-min CGM intervals).

- 🧪 **5 algorithms evaluated**: Simple/Multiple Linear Regression, kNN, LDA, QDA
- 📉 **GridSearchCV tuning** reduced RMSE to **32.60 mg/dL** and MAPE to **16.08%** — within the 15–20% clinical tolerance range
- 🔍 **Permutation importance** revealed basal insulin as the dominant glucose predictor; identified time-lag gap as the primary performance ceiling
- 🏆 **LDA achieved 96% classification accuracy** for glucose category detection (Low / Normal / High)
